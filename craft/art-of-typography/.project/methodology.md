# Methodology — The Art of Typography

## Research Question

What is the story of typography from Gutenberg's movable type to Google Fonts — and why does it matter beyond aesthetics?

## Approach

This project follows the paper-drive methodology: literature collection, source adequacy assessment, section-by-section deep-dives, and synthesis.

### Dimensions

Six axes structure the survey:

1. **History** — The 580-year arc from movable type to variable fonts
2. **Anatomy & Classification** — The technical vocabulary and systems for understanding type
3. **The Helvetica Question** — How one typeface became a proxy war for modernism vs expressionism
4. **Cognition** — What empirical research says about how letterforms affect the mind
5. **Digital Technology** — The format wars and democratization that brought type to everyone
6. **Culture** — How nations, movements, and marginalized communities encode identity through letterforms

### Source Strategy

- Historical references: encyclopedias, museum archives, design history blogs
- Academic studies: readability research, disfluency experiments, eye-tracking studies
- Design criticism: AIGA Eye on Design, Emigre, 99% Invisible, PRINT Magazine
- Industry publications: Fontfabric, Typofonderie, Typotheque, Blaze Type
- Technical documentation: CSS specifications, Google Fonts, font format specifications

### Quality Criteria

Each section required a minimum of 5 independent sources before a deep-dive was built. Sources were assessed for:
- **Recency**: Preference for post-2020 publications, with historical sources for foundational claims
- **Authority**: Academic publications, established design institutions, recognized practitioners
- **Specificity**: Sources that provide data points, dates, names, and verifiable claims
- **Diversity**: Multiple perspectives on contested topics (serif vs sans-serif readability, disfluency effect, Helvetica's legacy)

## Presentation Format

All deep-dives use the reading-assist template: sidebar navigation, light/dark theme toggle, expandable detail panels with serif prose, SVG diagrams, stat bars, timelines, and feature grids. The format prioritizes information density while maintaining readability through progressive disclosure (summary in main panel, full detail on demand).
