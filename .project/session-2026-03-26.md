## What Was Done This Session

### 1. Citation System — Wikipedia-style (repo-wide)
Established and retrofitted across **all 12 papers**:
```html
<sup class="fn"><a href="#ref-N">[N]</a><span class="tip"><a href="URL" target="_blank">Title →</a></span></sup>
```
- Hover `[N]` → tooltip with source title as clickable link (opens in new tab)
- Tooltip stays open when pointer enters it (`pointer-events: auto`)
- Click `[N]` → scrolls to consolidated References at bottom
- Font: `var(--sans)` not monospace
- No nested `<a>` tags — `[N]` link and tooltip link are siblings inside `<sup>`
- Per-section footers (`<div class="section-sources">`) fully eliminated

### 3. Theme Toggle Bug — Fixed Everywhere
CSS `[data-theme]` targets `<body>`. JS must use `document.body`, not `document.documentElement`.
Fixed in all 12 papers + science-of-humor index. Two JS patterns existed:
- `document.documentElement.setAttribute('data-theme', ...)` → `document.body.setAttribute(...)`
- `const root = document.documentElement;` → `const root = document.body;`

### 4. Heading Accent Word Rule
The `<em>` word = the **differentiator** (what makes THIS section unique), never the document's subject.
- ✗ The Neuroscience of *Laughter* (laughter = every section's subject)
- ✓ The *Neuroscience* of Laughter (neuroscience = this section's lens)
Fixed across: science-of-humor (4 files), retrocausal-emotion, religion-psychology.
