## Top Story

**OpenAI Launches GPT-5.4 with 1.05M Token Context and Dynamic Tool Search** · Product Launch

OpenAI released GPT-5.4, its most capable model to date, in three variants: Standard, Thinking (reasoning-first), and Pro (maximum capability). The context window extends to 1.05 million tokens. The architecturally notable addition is Tool Search — rather than requiring all tool definitions to be loaded into the prompt upfront, the model can dynamically look up relevant tool schemas at inference time. For complex agentic systems with large tool ecosystems, this is a meaningful cost and latency improvement. GPT-5.4 scores 83% on GDPval and 57.7% on SWE-Bench Pro, and reduces individual claim errors by 33% and full-response errors by 18% compared to GPT-5.2.

## Research Highlights

### CUDA Agent: Agentic RL for Automatic CUDA Kernel Generation
**Category:** Research Paper | **Source:** arXiv

From ByteDance Seed and Tsinghua University, CUDA Agent is a large-scale agentic reinforcement learning system that automatically generates optimized CUDA kernels. The system uses a three-level curriculum — progressing from simple element-wise operations through reduction patterns to complex multi-stage kernels like attention mechanisms — to generate 6,000 training examples without human annotation. The resulting kernels match or exceed hand-tuned implementations on several micro-benchmarks, pointing toward a future where GPU optimization is itself automated by AI.

[Read more](https://arxiv.org/abs/2603.11042)

### Spectrum: Training-Free Spectral Diffusion Feature Forecasting (CVPR 2026)
**Category:** Research Paper | **Source:** arXiv / CVPR 2026

Spectrum introduces a training-free method for accelerating diffusion model inference using Chebyshev polynomial-based spectral feature forecasting. By predicting future diffusion features in the spectral domain rather than recomputing them at every step, the method achieves up to 4.79× speedup on FLUX.1 and 4.67× on Wan2.1-14B with no measurable quality degradation on standard image generation benchmarks. Being training-free makes it drop-in compatible with existing diffusion pipelines — a significant practical advantage over distillation-based acceleration approaches.

[Read more](https://arxiv.org/abs/2603.10891)

## Agent & Tool Use

### GPT-5.4 Tool Search Unlocks Large-Scale Tool Ecosystems for Agents
**Category:** Product Launch | **Source:** OpenAI

The Tool Search feature in GPT-5.4 represents a meaningful architectural shift for agent builders. Today, loading a large tool registry into a model's context at query time is expensive and consumes tokens that could otherwise hold task-relevant information. Tool Search allows the model to dynamically retrieve only the tool schemas that are relevant to the current query — analogous to retrieval-augmented generation but applied to tool definitions rather than documents. This makes it practical to expose agents to tool ecosystems with hundreds or thousands of definitions without a proportional increase in per-query cost or latency.
