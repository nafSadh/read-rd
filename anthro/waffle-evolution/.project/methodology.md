# Methodology — waffle-evolution

## Research approach

This project follows the paper-drive methodology used across the read:rd anthro series.

### Source collection
- Web search sweep of 10+ queries across all 7 dimensions
- Sources assessed for credibility: Wikipedia (for factual scaffolding), Smithsonian Magazine, NPR, PBS Food, academic and food history publications
- Minimum 5–6 sources per section before deep-dive construction
- No fabricated citations — all URLs verified during search sweep

### Writing standards
- NPOV throughout — no first/second person authorial voice
- No editorial superlatives
- Contested origin stories noted as contested, not resolved arbitrarily
- Tensions documented as tensions, not resolved artificially

### HTML construction
- CSS variables and font stack match sibling anthro projects (religion-today, ice-cream-culture)
- Three-panel dd.html layout: left sidebar nav, center content, right source panel
- Survey and tensions papers: sidebar + scrollable paper-body layout
- Dark/light theme toggle consistent with sibling projects

### Limitations
- No access to paywalled food history journals (e.g., Gastronomica)
- Medieval primary sources (Le Ménagier de Paris, oublieurs guild records) accessed only via secondary sources
- Some origin stories (chicken & waffles, stroopwafel inventor) rest on oral tradition and cannot be fully verified
